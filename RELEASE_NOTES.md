# Gaia v7.0.1 Release Notes

This release introduces updates to three core dependencies: `tendermint` (v0.34.19),
`cosmos-sdk` (v0.45.3), and `tm-db` (v0.6.7). The `tendermint` dependency update
includes various p2p and consensus improvements. The `cosmos-sdk` release includes
minor patches and fixes. The `tm-db` release includes a `gorocksdb` update.
